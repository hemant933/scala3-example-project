# Spark Scala ETL Project

---

## 1. Project Overview

This ETL project demonstrates modern idiomatic Scala 3 features in a Spark-based data engineering environment. It leverages advanced language constructs such as union/intersection types, enums, contextual functions, and structural types, providing robust examples for building scalable, maintainable ETL pipelines.

**Tech Stack:**

| Component    | Version / Tool      |
|--------------|--------------------|
| Scala        | 3.x                |
| Spark        | 3.x                |
| Build Tool   | Maven (for Spark)  |
| Test Framework| MUnit              |

**Features:**

- Modular ETL pipeline (extract, transform, load)
- Strongly typed transformations and enums
- Contextual and explicit type control for safe data handling
- Examples of functional programming best practices with Spark
- Unit testing with MUnit
